\# 🎓 Student Grade Management System



This is a comprehensive C# console application designed to manage student records and track academic performance. This project serves as a practical implementation of Object-Oriented Programming (OOP) principles and was developed as part of my software development journey at Doğuş University and the Coursera Full-Stack Development program.



\## 🚀 Key Features

\- \*\*Student Management:\*\* Create and store student profiles with unique IDs.

\- \*\*Dynamic Grading System:\*\* Assign multiple grades and lessons to specific students.

\- \*\*Automated GPA Calculation:\*\* Real-time average calculation and letter grade assignment (AA, BA, etc.).

\- \*\*Interactive Interface:\*\* A user-friendly console menu driven by a robust `while` loop and `switch-case` logic.



\## 🏗️ Technical Implementation

\- \*\*Language:\*\* C# (.NET)

\- \*\*Concepts:\*\* Classes, Objects, Generic Lists (`List<T>`), Control Flow, and Input Validation.

\- \*\*Architecture:\*\* Structured with a focus on clean code and modularity.
